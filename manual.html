1. 이 사이트는 무엇인가요?

농촌관광 팸투어에 참석해주신 분들이 작성하신 SNS 홍보글(Instagram·블로그·X·TikTok 등)을 한 곳에 모으는 사이트입니다. 입력하신 정보는 주최측의 Google Sheets 에 자동 저장되어 나중에 컨텐츠별 보고서 작성에 활용됩니다.

📱 스마트폰에 최적화 되어 있어 PC·모바일 모두 사용할 수 있습니다. 별도 앱 설치 없이 브라우저에서 바로 작동합니다.

화면 구성

  • 신규 등록 — 처음 입력할 때 사용
  • 조회 / 수정 — 이름과 비밀번호로 본인 데이터 불러와서 수정·삭제
  • 사용 메뉴얼 — 지금 보고 계신 이 화면

2. 신규 등록 방법

1단계. [신규 등록] 화면에서 이름을 입력합니다.
2단계. 비밀번호를 설정합니다. 4자 이상 권장.
⚠️ 비밀번호는 나중에 본인 등록 내역을 조회·수정할 때 사용합니다. 반드시 기억해주세요.
3단계. 'SNS 게시물' 카드의 URL 입력란에 게시물 주소를 붙여넣습니다.
  • 예: https://www.instagram.com/p/CxXxXxX/
  • 예: https://blog.naver.com/abc/1234567
  • 반드시 http:// 또는 https:// 로 시작해야 합니다.
4단계. 해당 게시물에 노출된 컨텐츠를 모두 체크 합니다.
  • 4개 카테고리(저녁 바베큐파티 · 장소 · 선물 · 체험) 모두 처음부터 펼쳐져 있습니다.
  • 한 게시물에 여러 카테고리·항목이 노출됐다면 모두 체크 가능합니다.
  • 선택하면 각 카테고리 옆에 선택 개수 뱃지가 표시됩니다.
5단계. 게시물이 여러 개라면 입력항을 추가 합니다.
  • '추가 갯수' 에 숫자(1~20)를 입력하고 [+ 입력항 추가] 클릭
  • 입력한 숫자만큼 카드가 한 번에 추가됩니다.
  • 잘못 추가한 카드는 카드 우상단의 [삭제] 버튼으로 제거
  • 데이터가 입력된 카드를 삭제하면 확인창("입력된 값이 있습니다. 삭제하시겠습니까?") 이 표시됩니다.
6단계. 모든 입력을 마치면 [제출하기] 클릭. 완료 메시지가 표시되면 Google Sheets 에 저장된 것입니다.

3. 등록 내역 조회·수정

이전에 등록한 내용을 다시 보거나 수정·삭제할 수 있습니다.

1단계. 헤더의 [📋 내 등록 내역 조회/수정] 링크 클릭
2단계. 등록 시 사용한 이름·비밀번호를 입력하고 [조회하기] 클릭
⚠️ 비밀번호가 일치하지 않으면 조회되지 않습니다. 본인 데이터에만 접근 가능합니다.
3단계. 조회 성공 시 본인의 URL 카드들이 화면에 펼쳐집니다. 자유롭게 수정하세요:
  • URL 변경 / 컨텐츠 체크박스 변경
  • 새 카드 추가 (위 5단계와 동일)
  • 기존 카드 삭제
4단계. [수정 저장] 클릭 → 이전 데이터가 모두 지워지고 현재 화면의 데이터로 새로 저장됩니다.
5단계 (선택). 등록 자체를 취소하고 싶다면 [전체 등록 내역 삭제] 클릭
  • 확인창("등록 내역 전체를 삭제하시겠습니까? 이 작업은 되돌릴 수 없습니다.") 표시
  • [예] 누르면 본인의 모든 행이 시트에서 삭제됩니다.

4. 입력 규칙과 오류 메시지

제출/저장 버튼을 누를 때 입력값이 검증됩니다. 잘못된 항목이 있으면 화면 하단에 빨간 토스트 메시지가 표시되고 해당 입력란으로 자동 이동·강조됩니다.

오류 메시지원인해결
이름을 입력해주세요. 이름란이 비어있음 이름 입력
비밀번호를 입력해주세요. 비밀번호 누락 비밀번호 입력
비밀번호는 4자 이상 입력해주세요. 비밀번호가 3자 이하 4자 이상으로 변경
URL을 입력해주세요. URL란이 비어있음 URL 붙여넣기
SNS URL을 정확히 입력하세요. http:// 또는 https:// 로 시작하지 않음 완전한 URL 형식으로 입력
중복된 SNS URL을 입력하셨습니다. 여러 카드에 같은 URL 중복 카드 삭제 또는 URL 변경
노출된 컨텐츠를 1개 이상 선택해주세요. 체크박스 미선택 해당 게시물에 노출된 항목 체크
이름 또는 비밀번호가 일치하지 않습니다. 조회 시 본인 데이터 없음 등록 시 사용한 정확한 이름·비번 재입력
💡 오류가 발생하면 화면이 자동으로 잘못된 입력란으로 스크롤되고, 그 입력란에 빨간 테두리가 표시됩니다.

5. 노출 컨텐츠 항목

각 SNS 게시물에 노출된 컨텐츠를 4개 카테고리에서 모두 선택하세요.

🥩 저녁 바베큐파티

📍 장소

서도역-도시락원천체험휴양마을팬션까만집-다슬기탕 지리산허브벨리아티나 까페

🎁 선물

백향과젤리천연파스바로한끼4종 짜먹는들기름순창고추장사진인화

6. 자주 묻는 질문

비밀번호를 잊어버렸어요.
비밀번호 복구 기능은 없습니다. 다른 이름·비밀번호로 새로 등록하시거나, 주최측에 문의하시면 시트에서 직접 데이터를 조회·수정할 수 있습니다.
한 게시물에 여러 컨텐츠가 노출됐어요. 어떻게 입력하나요?
한 카드(=한 게시물 URL) 안에서 여러 카테고리·항목 체크박스를 동시에 선택하시면 됩니다. 카테고리 수만큼 카드를 나눠 입력할 필요 없습니다.
같은 SNS 계정에 게시물이 여러 개 있어요.
[+ 입력항 추가] 로 카드를 늘리고, 각 카드에 다른 게시물 URL 을 입력하세요. 같은 URL 을 두 번 입력하면 "중복된 SNS URL" 오류가 발생합니다.
컨텐츠 항목에 제가 노출한 게 없어요.
가장 가까운 카테고리에 체크하시거나, 주최측에 항목 추가 요청 부탁드립니다. (체크박스를 하나도 안 누르면 제출이 안 됩니다.)
등록 후 나중에 SNS 게시물 조회수가 늘면 어떻게 반영되나요?
조회수는 시트의 별도 컬럼(view_count) 으로 관리합니다. 주최측이 행사 종료 후 일괄적으로 수집·입력하며, 본인이 입력하실 필요는 없습니다.
제 데이터는 어디에 저장되나요?
Google Sheets (Submissions 시트) 에 자동 저장됩니다. 시트는 주최측 Google 계정 권한자만 접근 가능합니다.
한 번 제출한 뒤에 또 다른 게시물을 추가하고 싶어요.
두 가지 방법이 있습니다.
① [조회/수정] 로 들어가 본인 데이터를 불러온 뒤 카드를 추가해서 [수정 저장]
② [신규 등록] 으로 동일한 이름·비밀번호를 사용해 추가 게시물만 입력 후 제출 (두 제출이 모두 같은 본인 데이터로 묶입니다)
